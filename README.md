A mod installer/creator for Power Rangers: Battle for the Grid.
This is a modified version of SilentNightSound's GIMI (a version of 3DMigoto), edited to work with BFTG. 

Admittedly, this tool is abandoned. It still works for modding BFTG, and the documentation is insightful, but there likely won't be any changes made.
